Expected File Format

Use the monthly slot matrix exported from your scheduling sheet:

  • The first column lists each shift slot or time range (e.g., 8:00 AM - 5:00 PM).
  • Columns for Monday through Friday contain the agent names assigned to that slot for the week. Separate multiple names with line breaks, commas, or slashes.
  • Optional weekend columns are supported and will be imported when present.
  • Cells marked OFF, N/A, PTO, or left blank will be skipped.

Select the first and last Sundays for the month you are importing and the importer will generate individual daily schedules for every assigned agent across that span.
